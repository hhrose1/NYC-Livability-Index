HPD Violation Code Guide

Understand what each building violation code means

How to Check HPD Violations

1

Search

Go to HPD Online, check Complaints, Violations, and Bedbug Reports tabs

2

Assess

Review by class (A/B/C). Check how many are open vs. closed and how recent

3

Suspicious?

No violations at all? Cross-check NYC Open Data for missing reports

4

Research

Google the owner and property management company for lawsuits or complaints

Key Tips

  • Class A (minor): A few are normal for older buildings
  • Class B (hazardous): Multiple open = yellow flag
  • Class C (immediately hazardous): ANY open = red flag
  • Codes 729/730: Building chronically neglected — red flag
  • Codes 759/760/765: Vacate orders — illegal if rented
  • Age matters: Violations from 2001 matter less than one from 2025
