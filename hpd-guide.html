Key Tips

  • Class A (minor): A few are normal for older buildings
  • Class B (hazardous): Multiple open = yellow flag
  • Class C (immediately hazardous): ANY open = red flag
  • Codes 729/730: Building chronically neglected — red flag
  • Codes 759/760/765: Vacate orders — illegal if rented
  • Age matters: 2001 violations matter less than 2025 ones
